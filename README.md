# Nodejs-Application-with-Docker-CI
Description : 
Sample  node application that insert and update the user data using mongo and mongo-express docker images
1) Nodejs. 
2) Express framework. 
3) mongo official image .
4) mongo-express offical image using docker hub. 
5) docker composer file for all environments varibles and connecting both images on server.
6) Docker file for build docker imags.
